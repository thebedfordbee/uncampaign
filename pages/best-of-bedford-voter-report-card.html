Bedford Voter Report Card

Bedford Voter Report Card

A civic participation tool that lets Bedford voters look up their own public voter-history record, see a turnout score, and get a reminder for the next local election.

Voting is public participation. This just makes the participation easier to understand.

A Civic Participation Tool

Your Bedford Voting Record, in Plain English

This tool uses public voter-history records from the Board of Elections. It does not show who anyone voted for. It only reflects whether a voter participated in past elections and, when available, how the ballot was cast — such as Election Day, early voting, or absentee.

To protect privacy, the public voter file is not stored on this website. The lookup runs through a private Google Apps Script connection that returns only the matching voter's own report-card result. No personal lookup data is stored in your browser or passed through the URL.

How this was built

HOW WE BUILT THIS WITH AI

This campaign is committed to showing its work. When AI helps build something useful, it should be explained and made visible — not hidden.

HOW WE BUILT THIS WITH AI

This page was built as part of a deliberately low-cost local campaign experiment. Instead of buying campaign tech or hiring consultants, we used AI-assisted workflows to design a civic tool that helps residents understand their own public voter-history record.

The concept, copy, interface, and implementation were developed with AI assistance, then reviewed and refined by a human before publication. The voter data itself is handled through a private lookup process, not exposed as a downloadable public file on the website.

The prompt used to develop this tool:

Build a privacy-conscious voter report card tool for a local civic campaign. Use public voter-history records only to let residents look up their own participation record. Do not expose the raw voter file publicly. Return a simple turnout score, voting-method history, polling place, and election reminder. Keep the tone civic, transparent, and lightly self-aware.
