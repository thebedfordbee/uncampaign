# evidence/prompts/

Captures of the campaign's published AI instruction sets — the backbone of the "AI in the open" angle. The point is to show the prompts are genuinely public, not summarized.

## Capture list (from the outreach angle audit)

- [ ] **`/pages/ai` instruction-set screenshots** — at least the **Bedford Roundtable** instruction set and one other tool, showing the prompts are published in full.
- [ ] **Roundtable instruction set source** — a saved copy of `assets/js/roundtable-instruction-set.js` (or a screenshot of it rendered on the page).
- [ ] *(Optional)* a one-paragraph note on the human-review step, so the "human-reviewed, not AI-invented" claim is documented alongside the prompts.

## Rules

- Capture what is actually published on the site. Don't paraphrase a prompt into something cleaner than what's live.
- If a prompt changes, re-capture and date it.
