# evidence/coverage/

Screenshots and archived copies of **real, published** press coverage. Empty until coverage actually exists.

## Rules

- Add a file here **only** after a story has published. Pair it with a row in `/press-kit/coverage.md`.
- Name files `YYYY-MM-DD-outlet-slug.png` (e.g. `2026-06-01-record-review-uncampaign.png`).
- Where possible, also save the article text or a PDF/web-archive link, since links rot.
- Never place a mockup, draft, or anticipated story here. This folder is for things that ran.
