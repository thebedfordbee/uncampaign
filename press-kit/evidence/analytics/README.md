# evidence/analytics/

Dated snapshots of real engagement data. The site already runs **GA4** (`G-G7WVQR4MTH`) and **Microsoft Clarity** (`wis1l4ilnm`), so this data exists to be exported — it does not need to be invented.

## Capture list (from the outreach angle audit)

- [ ] **GA4 top-line** — sessions/users over a stated window, and top pages. Note the date range on the image.
- [ ] **Clarity heatmap or recording still** — optional, useful for showing how visitors use a tool.

## Rules

- Export real figures only. Never type an unverified metric onto a public page.
- State the date range on every snapshot.
- Redact anything that could identify an individual visitor.
- Use these to *substantiate* "people are using this" — not to claim a scale that the numbers don't support.
