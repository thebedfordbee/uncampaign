# evidence/quotes/

Shareable quote cards (image versions of lines from `quote-bank.md`) and, where useful, screenshots showing a quote in its original on-site context.

## Capture list

- [ ] **Quote cards** for the strongest published lines, e.g.:
  - "There is no Republican way to fill a pothole. There is no Democratic way to answer an email." (/pages/odds)
  - "AI helped build this campaign. Not to replace judgment — to reduce cost." (/pages/ai)
  - "You are allowed to buy bananas in peace." (/pages/rules)
- [ ] **Context screenshots** showing each quote on its source page (so reporters can verify it).

## Rules

- Only make cards from **[Published]** quotes, or from **[Suggested — for review]** quotes that Don has approved. Never turn unapproved draft language into a graphic that looks like a real quote.
- Use the site's design tokens (moss `#3E5240`, cream `#F8F5EF`, sage `#7B8C79`, Georgia serif) so cards match the campaign's look.
- Keep attribution and source page on each card.
