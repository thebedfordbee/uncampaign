# UNCAMPAIGN

A deliberately unconventional campaign repo for Don Scott's run for Bedford Town Supervisor in Bedford, New York.

This repository is the operating and build environment for The UNCAMPAIGN:

- ultra-low-cost
- AI-transparent
- creatively disciplined
- built for civic participation, not traditional persuasion
- designed to prove a local campaign can be useful, visible, and competitive without behaving like a normal campaign

## Core Principles

- Bedford deserves a choice
- Uncontested elections are unhealthy
- Choice over persuasion
- Transparency over theatrics
- Technology over money
- Conversation over campaigning
- Availability over intrusion
